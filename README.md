# hierarchy-display

## First steps
git clone project to your own machine
cd into folder

## Setup Project
```
npm install
```
### To run this application use
#### first make sure that the backend server is running
```
node src/server.js
```
#### second run the application

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
